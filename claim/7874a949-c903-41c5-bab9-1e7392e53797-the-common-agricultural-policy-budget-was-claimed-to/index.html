Omissions
  • The 45 billion euros comes from the mid-term review funds for rural areas within the existing MFF framework
  • The proposal involves reallocating funds from cohesion policy to agriculture, not creating new budgetary resources
  • This was a proposal by Ursula von der Leyen announced in January 2026, which would have been very recent relative to the session date of 2026-01-21
  • The funds are intended for the 2028-2034 MFF period, not the current CAP programming period
